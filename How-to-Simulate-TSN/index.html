一、OMNeT简介

OMNeT++(Objective Modular Network TestBed in C++) 是开源的基于组件的模块化的开放网络仿真平台，是近年来在科学和工业领域里逐渐流行的一种优秀的网络仿真平台。OMNeT++作为离散事件仿真器，具备强大完善的图形界面接口和可嵌入式仿真内核，同NS2等仿真平台相比，OMNeT++可运行于多个操作系统平台，可以简便定义网 络拓扑结构，具备编程，调试和跟踪支持等功能。OMNeT++ 主要用于通信网络和分布式系统的仿真，目前最高版本为5.7。

2.3 常见问题

  • 安装过程中不要使用root用户，会导致文件无法编译
  • 大多数教程中的环境变量设置与自己本机的环境变量不同，建议进入OMNeT文件夹后采用pwd指令查看本地具体路径
  • 配置过程中需要下载大量国外资源，建议配置APT代理和GIT代理

3.2 OMNeT++文件简介

OMNeT++中涉及多个文件，其中我们重点关注.ned .cc .ini .vec .sca文件。

  • .ned文件：（network description file）负责描述网络拓扑结构
  • .cc文件 ： 负责描述网络中的
  • .ini文件： 负责描述网络组件配置属性，如MAC地址，数据报文大小等
  • .vec文件： 统计网络流量，生成折线图
  • .sca文件： 统计网络流量，生成柱状图
