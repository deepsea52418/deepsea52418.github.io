How to use VPN

声明：本教程旨在技术分享！！！翻墙是为了学习！！！

首先介绍几个专有名词：

GFW = 国内的防火墙 梯子 = VPN节点 机场 = 提供多个VPN节点的商家 自建梯子 = 自己租国外的服务器搭建VPN节点

一、 VPN原理

VPN的原理很多文章都讲过了，在此不做赘述。实在想了解具体原理的可以看看下面这篇文章。

https://zhuanlan.zhihu.com/p/71536075

我简单讲一下我的理解，方便后面配置。

国内由于GFW的存在，无法访问国外的服务器（如Google等)。即从218.2.x.x到31.13.96.194被GFW阻拦。

图1 无VPN情况访问外网

而VPN的实现方式是让你的流量从218.2.x.x到127.0.0.1:7890(本地回环地址)，再到代理服务器，再到谷歌的服务器，这样就完成了访问。

图2 有VPN情况访问外网

我们这里只需要知道，我们电脑上要装一个代理软件（如ClashX），再搞定代理服务器，就可以翻墙了。

代理软件一般都是开源免费的，代理服务器是收费的。根据代理服务器的不同分为自建梯子和机场。自建梯子优点是便宜，缺点是麻烦，而且可选择性少；机场的优点是方便，全球各地的节点都可以选择，缺点是稍贵，而且担心机场跑路。

==本文选择机场的方式翻墙==

三、Windows配置

3.1 Windows系统配置

安装好ClashX后，软件界面如下，我们需要关注Port和System Proxy ,如果需要虚拟机翻墙，需要打开Allow LAN。

图4 ClashX配置

选择Profiles，在URL框中粘贴购买的机场订阅链接，点击下载。

图5 下载机场订阅链接

下载完成后，会在Proxies中出现很多节点，测试延迟后选择自己需要的节点，双击即可。

图6 选择需要的节点

全局代理表示所有流量均经过代理（你可以理解为你的电脑现在美国）

直连模式表示所有流量均不经过代理（你可以理解为你的电脑现在在国内）

规则模式表示按照一定的规则代理，在3.2节具体讲解

配置完成后打开百度搜索IP，看看自己的IP在不在国外。

图7 查询IP

至此Windows成功翻墙，可以正常访问Github等网站。

3.2 白名单配置

上文中所配置的属于全局代理，对于我们日常访问国内网站并不方便，为此我们需要配置白名单，选择性代理，即判断我们访问的服务器位于国内还是国外，对于国内的服务器采用直连模式，对于国外的服务器采用代理模式。

注：3.2节与3.3节目的都是实现白名单功能。3.2节属于全局，即类似QQ客户端的流量都走直连，youtube客户端的流量走代理；3.3节仅修改浏览器，即QQ客户端的流量还是会走代理，但是浏览器可以自动选择走代理还是直连

3.2节内容查考此链接.

3.4 无法联网情况

有时候开机发现无法访问互联网，大多数原因是上次关机前没有退出ClashX，导致网络代理本地环回。在设置->网络和Internet->代理中关闭使用代理服务器即可
